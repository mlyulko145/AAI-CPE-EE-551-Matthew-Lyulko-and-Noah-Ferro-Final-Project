# Grocery Value Analyzer

## Team Members
- Matthew Lyulko (mlyulko@stevens.edu)
- Noah Ferro (nferro@stevens.edu)

## Problem Description
This project analyzes grocery prices and necessity scores across different stores
to compute a composite quality score that helps identify the best value items.

## Program Structure
- `notebook/finalprojectmain.ipynb`: Main analysis and visualization
- `src/food_item.py`: FoodItem class
- `src/quality_metric.py`: QualityMetric class
- `src/utils.py`: Helper functions
- `data/`: CSV input datasets
- `tests/`: Pytest unit tests

## How to Run
1. Clone the repository
2. Open `notebook/finalprojectmain.ipynb`
3. Run all cells
4. (Optional) Run tests with:
   ```bash
   pytest
